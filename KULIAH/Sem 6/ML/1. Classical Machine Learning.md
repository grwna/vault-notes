# Definition
A program learns if it's performance (T) measured by a criteria (P) increases with experience (E). Learning agents uses *inductive reasoning* (generalizing data patterns) to learn information as opposed to being fed manually.

Scope of ML:
- Analyze complex data
- Identify hidden patterns
- Adaptive prediction
- Improve performance without explicit programming

Components of ML:
- Feature: attribute of data
- Target/Label: prediction output 
- Pattern: statistic relations of features and targets as found by the algorithm
- Training Set: data 
- Loss Function: measure of error/incorrectness of a model's prediction

## Reinforcement Learning
Doesn't require static dataset, but uses dynamic learning. Uses punishment-reward mechanism for learning.
Example: Traffic light automation, rewarded if traffic is smooth, punished if traffic jams.

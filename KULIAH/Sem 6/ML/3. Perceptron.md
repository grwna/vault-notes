# Introduction
Perceptron is inspired by neurons.

**Formal Definition**
Formally, perceptron is a linear threshold funtion that maps feature vectors to binary output.
$$y=\text{sign}(w_{0}+\sum_{j=1}^dw_{j}x_{j})$$
Sign - +1 if pos, -1 if neg or 0 (or sometimes a value of 0 is the result). The formula above shows the points around a hyperpane.
Perceptron's hyperplane is defined with
$$w_{0}+\sum_{j=1}^dw_{j}x_{j}=0$$

>[!note]
>Perceptron only works for binary and linearly separable data (workarounds exists however).

# Android Platform Architecture
From lowest to highest:
- Linux Kernel
- Hardware Abstraction Layer
- Native C/C++ libraries **or** Android Runtime
- Java API Framework
- System Apps

Each layers have their own purpose and protocols to communicate with other layers.
Also think about the multiplicity of each layer to other layers.

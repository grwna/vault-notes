# Introduction
## Key Points
- Model variation to optimize ensemble

## Why?
**Wisdom of the Crowds**
Many average people with different backgrounds and knowledge can beat a single expert.
There are 3 prerequisites of wisdom of the crowds, learn more.

Benefits of ensemble:
- Reduce Variance
- Reduce Bias
- Resistance against Noise

## Terminologies
- Parallel - models learn independently (which might or might not be parallel)
- Sequential - models learn in stages, next model fix the last model's mistake
- Homogenous - models use same algorithm 
- Heterogeneous - models can be of different algo
- Aggregation - combine prediction result (uses a model aswell)

## Parallel Ensembles
Independence - No error propagation 
Diversity can be achieved with:
1. Data variations -> sampling
2. Algorithm variations -> heterogeneous
3. Feature representations -> Random subspace

## Heterogeneous Parallels
- Weigthed voting (static/dynamic)
	- Dynamic voting includes confidence calculation
- Meta-learning/Stacking - this might seem like sequential, but it is really parallel

**Model Aggregation Techniques**
- Majority Voting (classification)
- Averaging (regression)
- Stacking (Meta-learning)
- Blending

# Bagging & Random Forests
## Key Points
- Bootstrap Sampling for diversity
- Majority voting for classification
- Bagging vs Random Forest
- OOB Score (bagging)
- Random feature selection improves random forest

## Bootstrap Sampling -> Bagging
- Bagging = Boostrap Sampling + Aggregating
- Randomly select samples: 
	- Selected samples: bag items
	- Non-Selected samples: OOB items (out of bag)
- Done to increase diversity since each models will use random samples

## Majority Voting
- Used as prediction aggregation 
- Predicted class probabilities is mean predicted class probabilities
## Random Forest
- Random Forest = Bootstrap Sampling + Randomized Trees
- Randomized decision trees
- Randomly select some features, as opposed to using all features

## Meta-Learning (using Stacking)
**Two Levels Meta-Learning**
- Base learners are trained on real dataset
- The predictions are aggregated into a level-one dataset
- Meta learners (usually logistic/lienar regression) are trained on the level-one dataset
- Meta learner with stacking is useeful for dynamic weighting
**Multi Level Meta-Learning**
- Each level added risks overfitting

## Gradient Boosting
- Gradient Boosting = Gradient Descent + Boosting
